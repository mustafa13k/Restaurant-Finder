# Restaurant-Finder
A Restaurant finder service which uses Zomato API to find collections of restaurants in a location.


It uses [Passport Authentication](https://github.com/jaredhanson/passport) with Local Strategy and [Connect-Mongo](https://github.com/jdesboeufs/connect-mongo) for which stores sessions in Mongo db to maintain persistence.

Use <b>_config.js_</b> to add your Zomato API Key.

## Installation

```bash
$ npm install
$ bower install angular
$ bower install angular-route

```

## Running

```
$ nodemon app.js
```
